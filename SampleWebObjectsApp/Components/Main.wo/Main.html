An integration wrapper is provided in this projet. Copy the Html2PDFService class into your project and set these 2 properties to start creating pdfs.

  • html2pdfServers=("http://localhost:1453/")

    Contain a list of server URLs, the code will try then is cascade in case of failure

  • html2pdfReadTimeout=15000

    The read timeout for the server request. The value is in ms and need to be enought for your larger PDFs
